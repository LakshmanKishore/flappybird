
# Flappy Bird

A singleplayer game. This is a cloned game of Flappy Bird. 
I have used circle as a bird and rectangles as the block.

## Built with

* HTML
* CSS
* Javascript

I learnt more about HTML5 Canvas and some Object Oeriented Concepts from Javascript.
